User Stories

User 1.1: The Student

As a college/ HS student I want a web application with music and visuals to keep me motivated studying without distracting me.

Elaboration: Users should be able to launch a cute 8-bit type of game that prompt a cozy virtual study space to maintain peace of mind when doing work or watching lectures.

Effort estimation: 12 hours for sole contributor

Performance test: Users should be able to launch the application from the Lexaloffle website and load the cartridge. The application should open and allow users to be able to have a character in a cozy study room.

2.1: Music lover

As a lo-fi enthusiast I want to listen to a different type of lo-fi beats as an alternative to YouTube with ads ruining my study vibe.

Effort Estimation: 9 hours for sole contributor

Performance test: A 30 minute playlist with music themes should be able to play as the application is launched

3.1: Aesthetic

As a college student the length of time I spend looking at my screen tends to wear out my eyes, I want to be able to study for a long time without the screen fatigue.

Elaboration: The user will be able to change theme, color mode, day-night theme, as well as the brightness of colors.

Effort Estimation 20 hours

Performance test: A successful change from day to night as well as the display should change when user prompts a darker theme or lighter theme
